실시간 지하철 위치를 알려주는 Metro Project 입니다.

# How to Start

```cmd
  npm run dev
  cd server
  npm run dev
```

# 기술스택

- frontend
  React, Typesript, Tailwind
- backend
  Nodejs

서울 열린 데이터 광장 실시간 지하철 API를 이용하였습니다.
