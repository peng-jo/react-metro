## 실시간 지하철 위치를 알려주는 Project 입니다.

# demo: https://react-metro.onrender.com/

# How to Start

```cmd
  npm install
  npm run dev
  cd server
  npm install
  npm run dev
```

# 기술스택

- frontend :
  React, Typesript, Tailwind
- backend :
  Nodejs

서울 열린 데이터 광장 실시간 지하철 API를 이용하였습니다.
